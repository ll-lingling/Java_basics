##  jQuery 

### 1 jQuery基本概念

#### 基本概念

 jQuery是一个javascript库，jQuery凭借着简洁的语法和跨平台的兼容性，极大的简化了js操作DOM、处理 事件、执行动画等操作。jQuery强调的理念是：'write less, do more'(写的少，做的多)。 

#### 基本特点

-   免费开源 
- 浏览器兼容性强 
- 轻量级 
- 链式编程 
- 插件扩展性强 (表单验证插件 日期插件 轮播图) 
- 写更少,做更多
